# RPG Battle Simulator

ターン制コマンドバトルを中心とした RPG ゲームシミュレーター。
キャラクターを強化しながら階層型ダンジョンを進み、長期的な育成と挑戦を提供するWebアプリケーションです。

## 主要機能

*   **戦闘**: サーバーサイドで計算される厳密なターン制バトル
*   **育成**: 経験値のストックと自由な割り振りシステム
*   **階層**: エンドレスに続くダンジョン（100階ごとの難易度上昇）
*   **マスタデータ**: DB管理されたキャラクター、スキル、アイテム、敵データ
